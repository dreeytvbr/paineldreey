# Painel Dreey

Instalação, atualização e ativação do painel XUI 1.5.13.

---

## 🚀 Instalação

Execute o comando abaixo em seu servidor:

```bash
wget "https://github.com/dreeytvbr/paineldreey/releases/download/MAIN/XUI_1.5.13.zip" -O /tmp/XUI_1.5.13.zip ; cd /tmp ; apt update ; apt install zip unzip -y ; unzip XUI_1.5.13.zip ; ./install
```

---

## 🔄 Atualização

Para atualizar uma instalação existente:

```bash
wget "https://github.com/dreeytvbr/paineldreey/releases/download/UPDATE/XUI_1.5.13_UPDATE.zip" -O /tmp/XUI_1.5.13_UPDATE.zip ; cd /tmp ; unzip XUI_1.5.13_UPDATE.zip ; ./update
```

---

## 🔓 Ativação (Crack)

Execute os comandos abaixo para ativar:

```bash
wget "https://github.com/dreeytvbr/paineldreey/releases/download/ATIVADOR/xui_crack.tar.gz" -O /tmp/xui_crack.tar.gz
```
```bash
cd /tmp
```
```bash
tar -xf xui_crack.tar.gz
```
```bash
sh /tmp/install.sh
```

---

## ⚠️ Requisitos

* Ubuntu 20/22
* Acesso root
* Conexão com a internet
* Pacotes `zip` e `unzip`

---

## 📌 Observações

* Faça backup antes de realizar atualizações.
* Utilize os comandos exatamente como apresentados.
* Certifique-se de que o servidor atende aos requisitos mínimos do sistema.
